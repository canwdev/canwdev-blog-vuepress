# уггСИђуФа С║єУДБWebтЈіуйЉу╗ютЪ║уАђ

СИЇтљїуАгС╗ХсђЂТЊЇСйюу│╗у╗ЪС╣ІжЌ┤уџёжђџС┐А№╝їТЅђТюЅУ┐ЎСИђтѕЄжЃйжюђУдЂСИђуДЇУДётѕЎ№╝їУ┐ЎуДЇУДётѕЎтЈФтЂџтЇЈУ««

HTTP тЇЈУ««Тў» TCP/IP тЇЈУ««ТЌЈСИГуџёСИђСИфтЇЈУ««

TCP/IP тѕєСИ║тЏЏт▒ѓ№╝џт║ћућет▒ѓсђЂС╝аУЙЊт▒ѓсђЂуйЉу╗ют▒ѓтњїТЋ░ТЇ«жЊЙУи»т▒ѓ

тѕєт▒ѓуџётЦйтцёТў»№╝џТ»ЈСИфт▒ѓтєЁжЃетЈ»С╗ЦУЄфућ▒Тћ╣тіе№╝їСИЇС╝џтй▒тЊЇтЁХт«Ѓт▒ѓ№╝ЏТ»ЈСИђт▒ѓуџёУ«ЙУ«АС╣ЪуЏИт»╣у«ђтЇЋ№╝їСИЇућеУђЃУЎЉтЁХт«Ѓт▒ѓсђѓ

сђљт║ћућет▒ѓсђЉ№╝џт║ћућет▒ѓтє│т«џС║єтљЉућеТѕиТЈљСЙЏт║ћућеТюЇтіАТЌХжђџС┐АуџёТ┤╗тіе№╝їТюЅСИђС║ЏтИИУДЂуџёт║ћућет▒ѓтЇЈУ««№╝џHTTPсђЂFTPсђЂDNS

сђљС╝аУЙЊт▒ѓсђЉ№╝џС╝аУЙЊт▒ѓТЈљСЙЏуйЉу╗юСИіСИцтЈ░У«Ау«ЌТю║С╣ІжЌ┤уџёТЋ░ТЇ«С╝аУЙЊсђѓТюЅСИцСИфТђДУ┤еСИЇтљїуџётЇЈУ««№╝џTCP№╝ѕС╝аУЙЊТјДтѕХтЇЈУ««№╝Ѕтњї UDP№╝ѕућеТѕиТЋ░ТЇ«ТіЦтЇЈУ««№╝Ѕ

сђљуйЉу╗ют▒ѓсђЉ№╝џуйЉу╗ют▒ѓ№╝ѕIPтЇЈУ««№╝ЅућеТЮЦтцёуљєуйЉу╗юСИіТхЂтіеуџёТЋ░ТЇ«тїЁ№╝їТЋ░ТЇ«тїЁТў»уйЉу╗юС╝аУЙЊуџёТюђт░ЈТЋ░ТЇ«С╝аУЙЊтЇЋСйЇ№╝їУ»Цт▒ѓУДёт«џС║єжђџУ┐ЄТђјТаиуџёУи»тЙёТііТЋ░ТЇ«тїЁС╝ажђЂу╗Ўт»╣Тќ╣

сђљТЋ░ТЇ«жЊЙУи»т▒ѓсђЉ№╝џжЊЙУи»т▒ѓућеТЮЦтцёуљєуйЉу╗юуџёуАгС╗ХжЃетѕє№╝їтїЁТІгТјДтѕХТЊЇСйюу│╗у╗ЪсђЂуАгС╗ХУ«ЙтцЄуџёжЕ▒тіесђЂуйЉтЇАсђЂтЁЅу║цуГЅуАгС╗Х

# IP тЇЈУ««

IPтЇЈУ««№╝ѕInternet Protocol№╝ЅуџёСйюућет░▒Тў»ТіітљёуДЇТЋ░ТЇ«тїЁС╝ажђЂу╗Ўт»╣Тќ╣№╝їУђїУдЂС┐ЮУ»Ђт«ъТЌХС╝ажђЂтѕ░№╝їжюђУдЂСИцСИфжЄЇУдЂТЮАС╗Х№╝џIPтю░тЮђтњїMACтю░тЮђсђѓIPжЌ┤уџёжђџС┐АСЙЮУхќMACтю░тЮђ№╝їСй┐ућеARPтЇЈУ««ТЅЇтЈ»С╗ЦтЄГтђЪMACтю░тЮђУ┐ЏУАїжђџС┐А№╝їСй┐ућеARPтЇЈУ««тЈ»С╗ЦТа╣ТЇ«жђџС┐АТќ╣уџёIPтю░тЮђТЪЦтЄ║т»╣т║ћуџёMACтю░тЮђсђѓ

# уггСИЅуФа HTTPТіЦТќЄСИГуџёHTTPС┐АТЂ»

У»иТ▒ѓуФ»№╝ѕClient№╝ЅуџёТіЦТќЄуД░СИ║У»иТ▒ѓТіЦТќЄ№╝їтЊЇт║ћуФ»№╝ѕServer№╝ЅуџёТіЦТќЄуД░СИ║тЊЇт║ћТіЦТќЄ№╝їHTTPТіЦТќЄТў»ућ▒тцџУАї№╝ѕСй┐ућеCR+LFТЇбУАї№╝ЅтГЌугдСИ▓ТќЄТюг

HTTPТіЦТќЄуџёу╗ЊТъё№╝џТіЦТќЄждќжЃе+уЕ║УАї№╝ѕCR+LF№╝Ѕ+ТіЦТќЄСИ╗СйЊ

тЈ»С╗ЦСй┐ућеFiddlerТЪЦуюІтјЪтДІТіЦТќЄтєЁт«╣

ТіЦТќЄСИ╗СйЊтњїт«ъСйЊСИ╗СйЊуџёти«т╝ѓ№╝џ

  1. ТіЦТќЄТў»HTTPжђџС┐АуџётЪ║ТюгтЇЋСйЇ№╝їжђџУ┐ЄHTTPжђџС┐АС╝аУЙЊ
  2. т«ъСйЊСйюСИ║У»иТ▒ѓТѕќтЊЇт║ћуџёТюЅТЋѕУййУЇиТЋ░ТЇ«УбФС╝аУЙЊ№╝їтЎетєЁт«╣ућ▒т«ъСйЊждќжЃетњїт«ъСйЊСИ╗СйЊу╗ёТѕљ

тјІу╝ЕС╝аУЙЊуџётєЁт«╣у╝ќуаЂ

жђџУ┐ЄТюЇтіАтЎетјІу╝ЕтљјтЈЉжђЂ№╝їт«бТѕиуФ»ТјЦТћХт╣ХУДБуаЂ

тИИућеуџётєЁт«╣у╝ќуаЂТќ╣т╝Ј№╝џgzip№╝ѕGNU№╝ЅсђЂcompress№╝ѕUNIX№╝ЅсђЂdeflate№╝ѕzlib№╝ЅсђЂidentity№╝ѕСИЇУ┐ЏУАїу╝ќуаЂ№╝Ѕ

тЈ»С╗ЦжђџУ┐ЄтѕєтЅ▓тЈЉжђЂуџётѕєтЮЌС╝аУЙЊу╝ќуаЂ№╝їУ«ЕТхЈУДѕтЎежђљТГЦТўЙуц║жАхжЮб№╝ЏУђїСИЇТў»тЁежЃеС╝аУЙЊт«їТѕљ№╝їУДБуаЂС╣ІтљјтєЇТўЙуц║жАхжЮбсђѓ

жђџУ┐ЄMIMEтњїMultipartТќ╣т╝Ј№╝їтЈЉжђЂтцџуДЇТЋ░ТЇ«уџётцџжЃетѕєт»╣У▒АжЏєтљѕ

УїЃтЏ┤У»иТ▒ѓ№╝џRange№╝їУ»иТ▒ѓСИђжЃетѕєуџёТќЄС╗ХтєЁт«╣

тєЁт«╣тЇЈтЋєУ┐ћтЏъТюђжђѓтљѕуџётєЁт«╣№╝їжђџУ┐ЄУ»иТ▒ѓТіЦТќЄТЪљС║ЏждќжЃетГЌТ«х№╝їТюЇтіАтЎеУЄфтіеУ┐ћтЏъТюђжђѓтљѕуџётєЁт«╣№╝ѕСЙІтдѓУ»ГУеђ№╝Ѕ

# уггтЏЏуФа У┐ћтЏъу╗ЊТъюуџёHTTPуіХТђЂуаЂ

# жђџС┐АТЋ░ТЇ«УйгтЈЉуеІт║Ј№╝џС╗БуљєсђЂуйЉтЁ│сђЂжџДжЂЊ

С╗БуљєтЈ»С╗ЦтЂџтѕ░т»╣ТїЄт«џURIуџёУ«┐жЌ«ТјДтѕХ№╝Џу╝ЊтГўС╗БуљєжђџУ┐Єт░єУхёТ║љу╝ЊтГўтѕ░С╗БуљєТюЇтіАтЎетіат┐Фт«бТѕиуФ»У«┐жЌ«жђЪт║д№╝ЏжђЈТўјС╗БуљєСИЇС╝џт»╣ТіЦТќЄУ┐ЏУАїС╗╗СйЋтіатиЦтцёуљє

уйЉтЁ│уџётиЦСйюТю║тѕХСИјС╗БуљєтЇЂтѕєуЏИС╝╝№╝їуйЉтЁ│УЃйСй┐жђџС┐Ау║┐Уи»СИіуџёТюЇтіАтЎеТЈљСЙЏжЮъHTTPтЇЈУ««уџёТюЇтіА№╝їСй┐ућеуйЉтЁ│УЃйТЈљжФўжђџС┐Ауџёт«ЅтЁеТђД

жџДжЂЊтЈ»ТїЅУдЂТ▒ѓт╗║уФІУхиСИђТЮАСИјтЁХС╗ќТюЇтіАтЎеуџёжђџС┐Ау║┐Уи»№╝їСй┐ућеSSLуГЅтіат»єУ┐ЏУАїжђџС┐А№╝їС┐ЮУ»Ђт«бТѕиуФ»СИјТюЇтіАуФ»уџёт«ЅтЁежђџС┐А

# уггтЁГуФа HTTPждќжЃе

ућеSSLу«ђтјєт«ЅтЁежђџС┐АС╣Ітљј№╝їУ┐ЏУАїуџёHTTPжђџС┐А№╝їуД░СИ║HTTPS№╝ѕУХЁТќЄТюгт«ЅтЁеС╝аУЙЊтЇЈУ««HTTP Secure№╝їТѕќ HTTP over SSL№╝Ѕ

тЈ»С╗Цт»╣ТіЦТќЄСИ╗СйЊУ┐ЏУАїтіат»є№╝їСйєт«бТѕиуФ»тњїТюЇтіАуФ»т┐ЁжюђТћ»ТїЂуЏИтљїуџётіат»єУДБт»єТю║тѕХ

SSLСИЇС╗ЁТЈљСЙЏтіат»єтцёуљє№╝їУђїСИћУ┐ўСй┐ућеС║єСИђуДЇУбФуД░СИ║У»ЂС╣дуџёТЅІТ«х№╝їтЈ»ућеС║јуА«т«џТќ╣

У»ЂС╣дућ▒тђ╝тЙЌС┐АС╗╗уџёуггСИЅТќ╣Тю║ТъёжбЂтЈЉ№╝їућеС╗ЦУ»ЂТўјТюЇтіАтЎетњїт«бТѕиуФ»Тў»т«ъжЎЁтГўтюеуџё№╝їС╝фжђаУ»ЂС╣дТў»т╝ѓтИИтЏ░жџЙуџёС║І

У»иТ▒ѓТѕќтЊЇт║ћтюеС╝аУЙЊУ┐ЄуеІСИГ№╝їжЂГТћ╗тЄ╗УђЁТІдТѕфт╣Ху»АТћ╣тєЁт«╣уџёТћ╗тЄ╗уД░СИ║СИГжЌ┤С║║Тћ╗тЄ╗

# HTTP+тіат»є+У«цУ»Ђ№╝ѕУ»ЂС╣д№╝Ѕ+т«їТЋ┤ТђДС┐ЮТіц = HTTPS

HTTPSТў»У║ФТіФSSLтЇЈУ««уџёHTTP№╝їУ┐љУАїтюет║ћућет▒ѓуџёSMTPтњїTELNETтЇЈУ««тЮЄтЈ»жЁЇтљѕSSLСй┐уће№╝їSSLТў»тйЊС╗іСИќуЋїСИіт║ћућеТюђт╣┐уџёуйЉу╗ют«ЅтЁеТіђТю»

HTTPSжЄЄућет»╣уД░уДўжњЦ№╝ѕтЁ▒С║ФуДўжњЦ№╝Ѕтіат»єтњїжЮът»╣уД░уДўжњЦ№╝ѕтЁгт╝ђуДўжњЦ№╝Ѕтіат»єТиитљѕуџётіат»єТю║тѕХ№╝їтюеуДўжњЦС║цТЇбуј»УіѓСй┐ућетЁгт╝ђуДўжњЦтіат»єТќ╣т╝Ј№╝їС╣Ітљјт╗║уФІуџёжђџС┐АС║цТЇбТіЦТќЄжўХТ«хСй┐ућетЁ▒С║ФуДўжњЦтіат»єТќ╣т╝Ј

Сй┐ућеТЋ░тГЌУ»ЂС╣дУ«цУ»ЂТю║Тъё№╝ѕCA№╝ЅУ»ЂТўјтЁгжњЦуџёТГБуА«ТђД

# уггтЁФуФа уА«У«цУ«┐жЌ«ућеТѕиУ║ФС╗йуџёУ«цУ»Ђ

уА«У«цуЎ╗тйЋУђЁТюгС║║уџёТќ╣т╝Ј№╝џт»єуаЂсђЂтіеТђЂС╗цуЅїсђЂТЋ░тГЌУ»ЂС╣дсђЂућЪуЅЕУ«цУ»ЂсђЂICтЇА

HTTPСй┐ућеуџёУ«цУ»ЂТќ╣т╝Ј№╝џBASICУ«цУ»Ђ№╝ѕтЪ║ТюгУ«цУ»Ђ№╝ЅсђЂDIGESTУ«цУ»Ђ№╝ѕТЉўУдЂУ«цУ»Ђ№╝ЅсђЂSSLт«бТѕиуФ»У«цУ»ЂсђЂFormBaseУ«цУ»Ђ№╝ѕтЪ║С║јУАетЇЋУ«цУ»Ђ№╝Ѕ

BASICУ«цУ»Ђ№╝ѕHTTP/1.0Ухи№╝Ѕ№╝їТў»ТюЇтіАтЎеСИјт«бТѕиуФ»С╣ІжЌ┤уџёУ«цУ»ЂТќ╣т╝Ј№╝їBASICУ«цУ»ЂтюеСй┐ућеСИіСИЇтцЪСЙ┐ТЇиуЂхТ┤╗№╝ѕТхЈУДѕтЎеТЌаТ│Ћт«ъуј░У«цУ»ЂТ│ежћђ№╝Ѕ№╝їУЙЙСИЇтѕ░тцџТЋ░уйЉуФЎТюЪТюЏуџёт«ЅтЁеуГЅу║Д№╝ѕУ┤дтЈит»єуаЂТўјТќЄС╝аУЙЊ№╝Ѕ№╝їтЏаТГцт╣ХСИЇтИИуће

DIGESTУ«цУ»Ђ№╝ѕHTTP/1.1Ухи№╝Ѕ№╝їТЈљСЙЏС║єжФўС║јBASICУ«цУ»Ђуџёт«ЅтЁеуГЅу║Д№╝їТЈљСЙЏт»єуаЂжў▓уфЃтљг№╝їСйєСИЇтГўтюежў▓ТГбућеТѕиС╝фУБЁуџёС┐ЮТіцТю║тѕХ№╝їСй┐ућеУїЃтЏ┤С╣ЪТюЅжЎљ

  1. ТюЇтіАуФ»тЈЉжђЂCertificate RequestТіЦТќЄ№╝їУдЂТ▒ѓт«бТѕиуФ»ТЈљСЙЏУ»ЂС╣д

жђџтИИSSLт«бТѕиуФ»У«цУ»ЂСИЇС╗ЁСЙЮжЮаУ»ЂС╣д№╝їУ┐ўС╝џтњїтЪ║С║јУАетЇЋУ«цУ»Ђу╗ётљѕтйбТѕљСИђуДЇтЈїтЏау┤аУ«цУ»ЂсђѓСй┐ућеУ»ЂС╣дуА«У«цт«бТѕиуФ»У║ФС╗й№╝їућеУАетЇЋТЮЦуА«У«цТў»ућеТѕиТюгС║║УАїСИ║

тЪ║С║јУАетЇЋуџёУ«цУ»Ђт╣ХСИЇТў»HTTPтЇЈУ««СИГт«џС╣Ѕуџё№╝їт«бТѕиуФ»С╝џтљЉТюЇтіАтЎеСИіуџёWebт║ћућеуеІт║ЈтЈЉжђЂуЎ╗тйЋС┐АТЂ»№╝ѕСИђУѕгСй┐ућеAJAXТѕќУђЁHTML Form№╝Ѕ№╝їТюЇтіАуФ»Webт║ћућетЪ║С║јУ«цУ»Ђу╗ЊТъюТЮЦтє│т«џТў»тљдУ«цУ»ЂТѕљтіЪ
