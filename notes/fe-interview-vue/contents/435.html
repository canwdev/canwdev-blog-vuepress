# vue怎么改变插入模板的分隔符？(delimiters) (opens new window)

在 Vue 初始化时配置：

# delimiters (opens new window)

  • 类型：Array

  • 默认值：[", "]

  • 限制：这个选项只在完整构建版本中的浏览器内编译时可用。

  • 详细：

    改变纯文本插入分隔符。

  • 示例：

    new Vue({
  delimiters: ['${', '}']
})

// 分隔符变成了 ES6 模板字符串的风格
